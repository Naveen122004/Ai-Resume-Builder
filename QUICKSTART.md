# Quick Start Guide

## Install & Run

```bash
npm install
npm start
```

## Usage Flow

### Step 1: Problem (Unlocked by default)
1. Read the problem statement
2. Copy artifact from Build Panel
3. Paste into Lovable
4. Click "It Worked" button
5. Upload screenshot (optional)
6. Next button unlocks

### Steps 2-8: Repeat Process
- Market → Architecture → HLD → LLD → Build → Test → Ship
- Each step unlocks only after previous step completed

### Final Submission
1. Complete all 8 steps
2. Visit `/rb/proof`
3. Enter Lovable, GitHub, and Deploy links
4. Click "Copy Final Submission"
5. Submit to instructor

## Key Features

✅ **No Skipping** - Linear progression enforced
✅ **Artifact Storage** - All uploads saved in localStorage
✅ **Progress Tracking** - Visual progress on proof page
✅ **Premium UI** - Professional build system layout

## Troubleshooting

**Next button disabled?**
- Make sure you clicked "It Worked" in Build Panel
- Screenshot upload is optional but recommended

**Lost progress?**
- Data stored in localStorage
- Clear browser cache will reset progress

**Want to restart?**
- Clear localStorage: `localStorage.clear()`
- Refresh page
